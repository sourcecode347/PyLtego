# PyLtego
PyLtego - A Domain OSINT Graph Tool

<img src="https://github.com/sourcecode347/PyLtego/blob/main/pyltego.png" style="width:100%;height:auto;"/>

# Description
PyLtego is a Python Script that you assign a Domain to and press Start Investigation and it identifies SubDomains, URLs, IPs and Emails in a graph. It is very simple to use and you can save each graph and reload it.

# Requirements
    pip install networkx matplotlib pillow requests beautifulsoup4 lxml

# Execute
    python pyltego.py
